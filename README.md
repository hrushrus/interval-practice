# Sophie and Chris' Interval Practice

A comprehensive, interactive web application for ear training and sight reading.

## Features
- **Multiple Training Modes**:
  - **Intervals**: Practice identifying distances between two notes.
  - **Chords**: Identify Major, Minor, Diminished, and Augmented triads.
  - **Scales**: Learn to recognize Major and various Minor scales.
  - **Perfect Pitch**: Challenge yourself to name notes by sound alone.
- **Audio & Visual**: Toggle between "Ear Training" (audio-only) and "Sight Reading" (musical staff).
- **Persistent Progress**: Your scores, current streak, and best streak are saved automatically in your browser.
- **Modern UI**: Clean, responsive design optimized for desktop and mobile.

## Built With
- **VexFlow**: For rendering beautiful musical notation.
- **Tone.js**: For high-quality piano audio samples.
- **Vanilla JavaScript, HTML5, & CSS3**.

## Local Development
1. Clone the repository.
2. Open `index.html` in your browser.
3. (Recommended) Run a local server to ensure audio assets load correctly:
   ```bash
   python3 -m http.server 8000
   ```
